# Implementation Plans

This directory contains phased implementation plans for the Jira MCP for Cursor project.

## Current Plan

**Active Plan:** `plan_jira_mcp_v1.md`  
**Version:** 1.0  
**Status:** Phase 1 in progress  
**Timeline:** 8 weeks (4 phases)

## Plan Structure

Each implementation plan follows this structure:

1. **Executive Summary** - Overview, objectives, success criteria
2. **Phase Overview** - High-level timeline and deliverables
3. **Detailed Phases** - For each phase:
   - Objectives and goals
   - Code and test deliverables
   - Technical tasks (day-by-day breakdown)
   - Acceptance criteria
   - Testing strategy
   - Security considerations
   - Dependencies and risks
   - Completion checklist

## How to Use This Plan

1. **Before starting a phase:** Review objectives and deliverables
2. **During implementation:** Follow day-by-day tasks, use TDD workflow
3. **Before completing a phase:** Check all acceptance criteria and completion checklist
4. **After phase completion:** Get approval before proceeding to next phase

## Workflow Integration

All implementation follows the workflow defined in `.cursor/00-workflow.mdc`:
- Branch strategy confirmation
- TODO tracking per phase
- Test planning and approval
- TDD cycle (Red → Green → Refactor)
- File critique (7-point review)
- Phase completion approval gates
